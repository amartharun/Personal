
# Tally Invoice Automation Pipeline

An AI-driven pipeline using the Google Gemini API to extract unstructured physical/digital invoice images and accurately map their financial summary data to match strict TallyPrime Accounting Invoice Excel import templates.

## Key Features
- Zero-deviation CGST/SGST summary row tracking.
- Automated Hamali charges mapping and postal pincode isolation.
- Balanced ledger matching ready for double-entry bookkeeping audits.

# Create a Virtual Enviroment
- python -m venv .venv
- .venv/Scripts/activate
- pip install -r requirements.txt
